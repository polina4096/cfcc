## Implemented features
### code generation
- function declarations
- function definitions
    - statements
        * compound (nested blocks)
        * return
        * goto & labels
        * expression
        * if
        * else
    - expressions
        * literal
        * function call (up to 4 arguments)
        * variable declaration initializer
        * variable access
        * variable assignment
        * subscript operator
        * math operators
            * addition
            * subtraction
            * multiplication
            * ~~division~~
        * boolean operators
            * relational
            * equality
            * logical
        * member access operators
            * address-of operator (only on locals, no arrays yet)
            * pointer dereference operator
- types
    * arrays
    * pointers
    * fundamental types
        * i32
        * ~~f32~~
